0 - поток активизирован по таймауту
240 - поток активизирован через Thread_Activate

(нужно учитывать, что Thread_Activate не меняет возвращаемое
в задачу значение; т.е. возвращаемое функцией значение, при активизации
потока через Thread_Activate, устанавливается заранее)


Thread_State

0 - поток готов к работе
1 - поток ожидает семафор
2 - поток ожидает событие
3 - поток ожидает сигнал
4 - поток ожидает какое-то событие
15 - поток заблокирован
16 - поток ожидает окончания таймаута
17 - поток ожидает семафор с таймаутом
18 - поток ожидает событие с таймаутом
19 - поток ожидает сигнал с таймаутом
20 - поток ожидает какое-то событие с таймаутом
31 - поток заблокирован с таймаутом


Mutex_Get

1 - успешный захват мьютекса
3 - повторный захват мьютекса
4 - взаимная блокировка
240 - поток активизирован через Thread_Activate


Mutex_TryGet

0 - мьютекс захвачен другой задачей
1 - успешный захват мьютекса
3 - повторный захват мьютекса


Mutex_GetWithTimeout

0 - поток активизирован по таймауту
1 - успешный захват мьютекса
3 - повторный захват мьютекса
4 - взаимная блокировка
240 - поток активизирован через Thread_Activate


Mutex_Owner

Возвращает номер потока, владеющего мьютексом (0 -- мьютекс свободен)


Sleep

0 - поток активизирован по таймауту
240 - поток активизирован через Thread_Activate



Signal_Set

Возвращает 0, если данный сигнал никто не ожидал
Возвращает не 0, если данный сигнал ожидал хотя бы один поток


Signal_IsSet

Возвращает не 0, если сигнал активен (установлен)
Возвращает 0, если сигнал сброшен (не активен)


Signal_Wait

1 - дождались сигнала
3 - сигнал уже есть (т.е. сигнал был активен уже в момент вызова Signal_Wait)
240 - сигнала нет, поток активизирован через Thread_Activate


Signal_WaitWithTimeout

0 - поток активизирован по таймауту
1 - дождались сигнала
3 - сигнал уже есть
240 - сигнала нет, поток активизирован через Thread_Activate



Event_Occur

Возвращает 0, если данное событие никто не ожидал
Возвращает не 0, если данное событие ожидал хотя бы один поток


Event_Watch

1 - дождались события
240 - события не дождались, поток активизирован через Thread_Activate


Event_WatchWithTimeout

0 - поток активизирован по таймауту
1 - дождались события
240 - поток активизирован через Thread_Activate



Semaphore_Up

Возвращает новое состояние семафора


Semaphore_Down

1 - семафор открыт
240 - семафор недоступен (закрыт), поток активизирован через Thread_Activate


Semaphore_TryDown

0 - семафор недоступен (закрыт)
1 - семафор открыт


Semaphore_DownWithTimeout

0 - семафор недоступен (закрыт), поток активизирован по таймауту
1 - семафор открыт
240 - семафор недоступен (закрыт), поток активизирован через Thread_Activate
